SITIO WEB

Un sitio web es una colección de páginas en internet que están relacionadas entre sí y que comparten una dirección web única. Es un espacio virtual donde la información, el contenido y los servicios se presentan y se hacen accesibles para los usuarios a través de un navegador web.
